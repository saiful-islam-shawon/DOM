some important event

🔥 1. Click Event

👉 সবচেয়ে বেশি ব্যবহার হয়

btn.addEventListener(“click”, function () {
console.log(“Button clicked”);
});

📌 যখন user button বা element-এ click করে তখন trigger হয়।

⌨️ 2. Key Events

👉 keyboard input handle করার জন্য

keydown

document.addEventListener(“keydown”, function (e) {
console.log(e.key);
});

📌 key press করলেই run হয়।

🖱️ 3. Mouse Events

mouseover

btn.addEventListener(“mouseover”, function () {
console.log(“Mouse over button”);
});

mouseout

btn.addEventListener(“mouseout”, function () {
console.log(“Mouse left button”);
});

📌 hover effect বানাতে কাজে লাগে।

📥 4. Input Event

👉 form input track করার জন্য

const input = document.querySelector(“input”);

input.addEventListener(“input”, function (e) {
console.log(e.target.value);
});

📌 user টাইপ করলেই value change detect করে।

📄 5. Submit Event

const form = document.querySelector(“form”);

form.addEventListener(“submit”, function (e) {
e.preventDefault();
console.log(“Form submitted”);
});

📌 form submit হলে trigger হয়।

🌐 6. DOMContentLoaded

document.addEventListener(“DOMContentLoaded”, function () {
console.log(“Page fully loaded”);
});

📌 পুরো HTML load হওয়ার পরে run হয় (best practice 🔥)

📦 Bonus: Change Event

const select = document.querySelector(“select”);

select.addEventListener(“change”, function () {
console.log(“Value changed”);
});

⚡ Pro Tip

👉 onclick না ব্যবহার করে addEventListener() use করো
কারণ এটা বেশি flexible এবং professional way।
